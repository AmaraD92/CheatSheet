There may be some fonts that are not websafe. In order to load them, we have to do the following:

  1. Add font link with a tag or import it to the stylesheet
  2. Set the typeface in their respective tag
  3. Set fallback fonts

Tag Name Description
< Table > Table The wrapper element for all HTML tables
< thead > Table Head The rows defining the column headers in a table
< tbody > Table Body The set of rows containing actual table data
< tr > Table Row The table row container
< td > Table Data The table row container
< tfoot > Table Foot The set of rows defining the footer in a table
